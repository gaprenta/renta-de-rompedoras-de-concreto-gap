# GAP Renta de Rompedoras 🏗️

> **Especialistas en demolición y equipo para construcción.**

Este repositorio contiene los archivos de la página web oficial de **GAP Renta de Rompedoras**, diseñada para ofrecer información técnica, catálogo de equipos y contacto directo para servicios en el área metropolitana de Monterrey.

---

## 🛠️ Nuestro Catálogo
Contamos con equipos de alto rendimiento para diversas necesidades de obra:

* **12kg:** Ideal para retiro de azulejo y trabajos ligeros en muros.
* **15kg:** Versatilidad para demoliciones medianas.
* **30kg:** Potencia industrial para losas de concreto y cimentaciones pesadas.

**Marcas destacadas:** Bosch, Makita, DeWalt, Edoevom.

---

## 🌐 Tecnologías Utilizadas
Para el desarrollo de este sitio se implementaron:
* **HTML5 / CSS3:** Estructura y diseño con una estética industrial (Amarillo/Negro).
* **GitHub Pages:** Hosting y despliegue del sitio.
* **SEO Optimo:** Configuración para visibilidad en Google Business y Search Console.

---
